# Scripts

## reset_db.py
Reinicializa la base de datos con datos de demostración.

### Usage
```bash
python scripts/reset_db.py
```

### What it does
- Limpia la base de datos
- Inicializa las tablas
- Carga 5 rendiciones de demostración
- Carga 4 notificaciones de demostración

### When to use
- Primera configuración
- Para resetear datos
- Para pruebas limpias
